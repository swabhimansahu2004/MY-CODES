HTML Tags -Tags occur in pairs.The opening tag is known as start tag and the closing tag is known as end tag. Example:- This is a Bold Tag

HTML Elements-The Whole Line consisting of tags,including attributes and the Content inside them is refered as Elements. Without Content an Element is known as Empty Element. HTML elements support nesting which means we can put 1 element is other elements. Example:-p,a,div,etc.

Attributes- They are used about how a element should be Displayed.They are always defined in the opening tag.

Event Handler Content Attributes-These attributes empower HTML tags to invoke a script when a certain “event” occurs.In an HTML code, we can add an event handler attribute to an HTML element to capture an event for a specific user action.

Head-The head element is the header section of an HTML document. This element contains other tags like title, meta, script, and style used for describing metadata information of the page. Each tag has a specific purpose. The content of these tags is not visible on the page; the only tag whose content is visible is the title tag that puts the content on the browser title bar.
