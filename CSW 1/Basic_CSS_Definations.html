CSS(Cascading Style Sheets)-It falls into domain of style sheet languages.It is Primarily used for defining the presentation of web pages while HTML defines the Structure of aa Web Page. The CSS Rules are designed from:- 1.Properties-Properties are Configured with values to modify how the Content of Html should be displayed to the user. 2.Selector-It chooses the elements by their IDs,names,types,attributes,etc.,on which desired style is going to be applied.

DOM Model(Document Object Model)=This model is used by the computer's memory for representation of the document. The DOM model consists of a structure which resembles a tree data structure.All the attributes, elements, and text of the markup language are added as the DOM node in this structure. DOM describes how the content of a document and CSS have to be linked.
