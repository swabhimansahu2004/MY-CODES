BootStrap 4 Jumbotron

A Jumbotron indicates a big grey box for calling extra attention to some special contents or information.


Jumbotron Example

This is used to highlight importance of a piece of content.


Use a "div" element with class ".jumbotron" to create a Jumbotron.

Jumbotron Fluid Example

It uses utility classes for typography and spacing to space content out within the larger container.


Use a "div" element with class ".jumbotron jumbotron-fluid" to create a Jumbotron.
